На сервере хранится очередь из 4 заданий. Задания из себя представляют класс WorkItem, в котором рандомно генерируются коэффициенты для решения системы уравнения типа:
A1*x+B1*y = C1
A2*x+B2*y = C2
У каждого из четырех заданий есть состояние "решенности". За это отвечает соответствующий flag.
Если задание нерешенное, то при подключение клиента к серверу ему достается это задание.
Когда все четыре задания будут решены, то сервер выдаст сообщение о необходимости перезапуска сервера, для генерации новой очереди заданий.
